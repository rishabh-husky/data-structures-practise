## a stack is LIFO

Last in,First Out.

# Stack  operations

### Implement api for stack

- Push(Key)

add Key into collection

> O(n):1

- Key Top()

returns most recently-added key

> O(n):1

- Key Pop()

removes and returns most recently-added key

> O(n):1

-[Boolean Empty()

is collection empty

> O(n):1
